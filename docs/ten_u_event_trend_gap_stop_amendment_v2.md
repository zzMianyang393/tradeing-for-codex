# v2 硬止损跳空成交修正

旧回放在小时 K 线触及硬止损时，总是假设止损市价单按计划止损价成交。若该小时开盘已经越过止损，
真实成交不可能回到更优的止损价，这会低估小币种的尾部损失。

修正后的冻结规则：

- 多单硬止损原始成交价为 `min(计划止损价, 当前小时开盘价)`；
- 空单硬止损原始成交价为 `max(计划止损价, 当前小时开盘价)`；
- 随后继续按冻结成本模型扣除 0.02% 出场滑点和 0.05% taker 手续费；
- 非跳空的盘中触发仍按计划止损价成交；
- 结构退出和 48 小时退出不受影响。

修正发生时前瞻账本为 0，未访问前瞻收益，也未改变信号或风险参数。历史密封筛选账户结果逐字段
保持不变，说明其中唯一 LAB 硬止损并非开盘跳空。本规则只会令未来跳空止损的亏损更差或相同，
不会产生更好的回测成交。
