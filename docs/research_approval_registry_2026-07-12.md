# 研究结论批准登记（2026-07-12）

## 目的

`research_approval_registry.py` 将研究结论写成机器可读清单。它解决的不是找收益，
而是防止旧高收益、前视偏差报告或已淘汰分支被重新当作可交易策略。

## 使用

```bash
python research_approval_registry.py --out reports/research_approval_registry.json
```

该登记不修改 EA 核心。运行时安全保护仍由 `runner.py` 负责，研究批准由本登记提供
可追溯决策依据。
