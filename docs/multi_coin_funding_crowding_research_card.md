# 多币种 Funding 拥挤反转：预注册研究卡

## 假设

当 OKX 永续市场中多个币种的资金费率同时达到极端正值时，表明市场多头过度拥挤。
这种跨币种的同步极端状态通常不可持续，后续倾向于反转或回归均值。

**与已淘汰的 Funding Carry 的区别：**
- Funding Carry：赚取资金费率收入（现货多 + 永续空），依赖费率覆盖成本
- 本方向：将 funding 作为情绪信号（拥挤度指标），预测价格方向变化
- 本方向不收取 funding，仅用 funding 极端程度作为入场信号

## 冻结规则

- 数据：OKX 永续资金费率归档（25 币种，590+ 天），OKX 15m OHLCV（28 币种）。
- 仅研究正 funding 极端：负 funding 方向的借币成本无可靠历史。
- 联合信号：每日 3 次结算后，计算所有币种的 funding 均值和极端币种占比。
- 触发：跨币种 funding 均值 >= 历史 90 分位，且极端币种占比 >= 50%。
- 入场：信号确认后的下一根 15m 开盘做空（或不入场，仅做事件研究）。
- 持有：16 根 15m K 线（4 小时）。
- 成本：双边 taker fee 0.05% + 滑点 0.03%，往返 0.16%。
- 形成样本：截至 2025-07-10 的 365 天。

## 通过门槛

形成期先做事件研究，且必须同时满足：

1. 至少 15 个事件；
2. 总体净收益均值为正（扣 0.16% 往返成本后）；
3. 净胜率 >= 55%；
4. 不由单一月份或单一极端日期贡献超过 30% 的事件；
5. 冻结规则后，才运行样本外和组合回测。

不满足任一条件即淘汰，不调整分位数阈值、持有期或入场方向。

## 可证伪性

- 若跨币种 funding 极端后价格无显著反转，假设不成立。
- 若反转幅度 < 0.16%（往返成本），假设不可交易。
- 若仅在特定月份有效（如牛市），假设无泛化能力。
