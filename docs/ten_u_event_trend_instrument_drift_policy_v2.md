# v2 合约规格漂移与退市策略

每次小时刷新在写入市场数据前重新读取 OKX 合约规格，并与研究开始时冻结的规格比较。

以下变化会使刷新立即失败：

- 合约从 `live` 变为暂停、预上线或退市状态；
- `ctVal`、`ctValCcy`、`lotSz`、`minSz` 或结算币发生变化；
- 交易所允许的最高杠杆低于策略冻结的 3 倍上限；
- 合约 ID 与冻结币种不一致。

数字字符串会先按十进制数比较，因此 `10` 与 `10.0` 不构成假漂移。检测到真实漂移时，v2 不会：

- 使用新规格重算旧交易；
- 静默继续积累错误仓位样本；
- 删除该币后只报告剩余赢家；
- 换入另一个近期热门币种。

该版本将停止并等待一个新的、独立预注册研究版本。这样可避免合约规格变化和退市造成的幸存者偏差。
