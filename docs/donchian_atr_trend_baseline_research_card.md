# 唐奇安 + ATR 趋势基准：预注册研究卡

## 家族定义

本研究卡统一代表以下同一机制家族的策略：

- 海龟交易（Donchian 突破 + ATR 跟踪止损）
- N 日高低点突破（通道突破）
- ATR 波动率突破
- 趋势跟踪 + ATR 止损

**核心机制：** 价格突破 N 日高点做多 / N 日低点做空，止损为 M × ATR。

## 预注册参数窗口

| 参数 | 值 | 说明 |
|---|---|---|
| 突破窗口 | 20 日（Donchian 20） | 行业标准，不做网格 |
| ATR 周期 | 14 日 | 行业标准 |
| ATR 止损倍数 | 2.0 × ATR | 海龟标准 |
| 持有窗口 | 最多 10 日（96 × 10 = 960 根 15m） | 或触发止损 |
| 入场 | 突破确认后下一根 15m 开盘 | 不用同一根 K 线 |
| 出场 | 止损触发 或 持有到期 | 不用追踪止损 |

**不得扫描的参数：** 突破窗口（10/20/50）、ATR 倍数（1/2/3）、持有期（5/10/20 日）、
币种集合、杠杆倍数。上述值为冻结值。

## 数据

- OKX 15m OHLCV（28 币种，年度覆盖）
- 成本：双边 taker fee 0.05% + 滑点 0.03%，往返 0.16%
- 不使用任何外部数据

## 形成期 / OOS 分割

| 阶段 | 范围 | 用途 |
|---|---|---|
| 形成期 | 2024-01-01 至 2024-12-31 | 事件审计 + 通过门槛 |
| 样本外 | 2025-01-01 至 2025-07-10 | 冻结规则后验证 |

## 通过门槛（形成期）

1. 至少 50 个去重交易事件（跨币种合计）；
2. 扣成本后净收益均值为正；
3. 净胜率 >= 45%（趋势跟踪策略通常胜率较低，但盈亏比高）；
4. 盈利因子（总盈利 / 总亏损）>= 1.2；
5. 不由单一币种贡献超过 30% 的总收益；
6. 不由单一月份贡献超过 25% 的事件。

## 样本外验证（形成期通过后）

1. 样本外净收益均值必须为正；
2. 样本外盈利因子 >= 1.0；
3. 样本外胜率 >= 40%。

## 失败后不得做的事

- 不改突破窗口（10/50/100）
- 不改 ATR 倍数（1/1.5/3/4）
- 不改持有期
- 不加过滤条件（RSI、成交量、趋势过滤等）
- 不缩币种集合（仅 BTC、仅高流动性等）
- 不改方向（做空变做多）
- 不改止损机制（移动止损、固定止损等）

## 适用行情

本策略理论上适用于**趋势行情**。震荡行情中会频繁假突破导致亏损。
可使用 `regime_validation.py` 的行情标签在事后审计中拆分表现，
但**不得**在形成期用行情标签过滤信号（这等同于事后选择）。

## 与其他方向的隔离

- 与已淘汰的 `volatility_compression_breakout` 不同：后者使用 ATR 压缩 + 成交量确认
- 与已淘汰的 `low_turnover_trend` 不同：后者使用换手率过滤
- 本方向是纯价格突破 + ATR 止损，不含成交量或换手率条件

## 研究流程

1. 先写独立审计脚本（不接入 `candidate_strategies.py`）
2. 输出形成期事件报告
3. 通过门槛后，冻结规则
4. 运行样本外验证
5. 全部通过后，才可接入 `unified_validation.py`

## 可证伪性

- 若突破后价格不延续（均值回复为主），假设不成立
- 若成本吞噬大部分利润，假设不可交易
- 若假突破率过高（胜率 < 30%），假设不适用于当前市场
