# 低换手候选研究卡模板

## 使用说明

任何未来低换手方向在申请研究卡时，必须填写以下全部字段。
缺少任何字段的研究卡不予审查。

---

## 1. 基本信息

- **研究 ID：** （英文短标识，如 `low_freq_macro_trend`）
- **中文名：** （如"低频宏观趋势跟踪"）
- **假设：** （一句话描述市场机制偏差）

## 2. 数据来源

- **数据类型：** （如 OKX 15m OHLCV / funding / 日线 OI）
- **免费可复现性：** （是/否，说明来源 URL）
- **OKX 同源性：** （是否使用 OKX 同合约数据）
- **覆盖天数：** （至少 365 天）
- **字段覆盖率：** （关键字段 ≥ 95%）

## 3. 持有周期

- **预期持有周期：** （必须 ≥ 3 天，即 ≥ 288 根 15m K 线）
- **理由：** （为什么选择这个持有周期）

## 4. 事件频率

- **预期每月事件数：** （必须 ≤ 12）
- **预期总事件数（形成期）：** （至少 30）
- **理由：** （为什么事件频率如此低）

## 5. 执行成本

- **执行腿数：** （单腿/双腿/四腿）
- **单次往返成本：** （至少 0.16% 单市场往返，0.32% 四腿往返）
- **月度成本估计：** （事件数 × 单次成本）

## 6. 与已淘汰家族的隔离

- **最相似的已淘汰方向：** （如 `low_turnover_trend`）
- **为什么不是小变体：** （说明机制差异，不是参数差异）
- **注册表中已有哪些类似方向：** （列出）

## 7. 信息可用时间

- **信号数据可用时间：** （如"日度 OI 16:00 UTC"）
- **入场价格：** （信号确认后的下一根 15m 开盘）
- **是否存在前视偏差风险：** （是/否，说明如何避免）

## 8. 形成期 / OOS

- **形成期：** （如 2024-01-01 至 2024-12-31）
- **样本外：** （如 2025-01-01 至 2025-07-10）
- **分割理由：** （为什么选择这个分割点）

## 9. 通过门槛（形成期）

1. 至少 ___ 个事件
2. 净收益均值 > 0
3. 净胜率 ≥ ___%
4. 盈利因子 ≥ ___
5. 单币贡献 ≤ ___%
6. 单月集中度 ≤ ___%

## 10. 样本外验证

1. 净收益均值 > 0
2. 胜率 ≥ ___%
3. 盈利因子 ≥ ___

## 11. 失败后不得调整的参数

- [ ] 阈值/分位数窗口
- [ ] 持有期
- [ ] 止损/止盈倍数
- [ ] 币种集合
- [ ] 方向（做多/做空）
- [ ] 行情标签过滤
- [ ] 成本假设
- [ ] 其他：___

- [ ] 不接入 `candidate_strategies.py`
- [ ] 不接入模拟盘或实盘
- [ ] 形成期通过前不接入 `unified_validation.py`

## 审查清单

提交研究卡前确认：

- [ ] 所有 13 个字段已填写
- [ ] 持有周期 ≥ 3 天
- [ ] 每月事件 ≤ 12
- [ ] 数据免费可复现
- [ ] OKX 同源
- [ ] 信息可用时间已明确
- [ ] 入场价格为下一根 15m 开盘
- [ ] 与已淘汰方向有机制差异（不是参数差异）
- [ ] 失败后不得调整的参数已列出
- [ ] 禁止接入已确认
