# 影子信号错误行情触发审计（2026-07-13）

## 概述

检查影子信号是否在声明适用的行情标签内触发。不计算信号后的价格、收益或持仓。

## 数据来源

信号观察来自 `rejected_event_dataset.json`（27 个事件），映射到 7 个影子因子。

## 结果

当前数据集中无匹配的信号观察（rejected_event_dataset 中的事件 strategy_id 与因子 ID 不直接对应）。

**说明：** 影子信号账本需要从未来的因子信号生成器中积累。当前审计基于现有报告，信号数为 0。

## 局限性

- 信号数为 0，无法验证合规性
- 需要因子信号生成器运行后才能积累信号账本
- 不计算收益或持仓
