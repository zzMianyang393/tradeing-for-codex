# 行情标签覆盖与稳定性审计（2026-07-13）

## 概述

对 8 个币种（BTC/ETH/SOL/DOGE/LINK/ADA/AVAX/DOT）的已完成 4h 行情标签进行覆盖与稳定性审计。

## 标签定义

基于已完成 4h K 线（EMA20/50/200 + ATR%）：
- **趋势上行**：EMA20 > EMA50 > EMA200 且 trend_strength ≥ 1.0
- **趋势下行**：EMA20 < EMA50 < EMA200 且 trend_strength ≤ -1.0
- **高波动转换**：ATR% ≥ 3%
- **震荡**：以上均不满足

标签可用时间：bar_ts + 4h（4h K 线收盘后）
